# dotfiles

Configurazione personale del terminale macOS (Apple Silicon).

## Contenuto
- `zshrc` — configurazione Zsh con zinit
- `gitconfig` — configurazione Git con delta e GPG
- `starship.toml` — prompt Starship con tema Catppuccin

## Setup
```bash
cp zshrc ~/.zshrc
cp gitconfig ~/.gitconfig
cp starship.toml ~/.config/starship.toml
source ~/.zshrc
```
